Markdown 完整常用语法清单

一、基础文本格式

1. 标题（层级结构）

层级 源码写法 渲染效果 HTML 对应
一级 # 一级标题（# 后空格） # 一级标题（最大号加粗） <h1>
二级 ## 二级标题 ## 二级标题（大号） <h2>
三级 ### 三级标题 ### 三级标题（中号） <h3>
四级 #### 四级标题 #### 四级标题（小号） <h4>
五级 ##### 五级标题 ##### 五级标题（更小） <h5>
六级 ###### 六级标题 ###### 六级标题（最小） <h2>

小贴士： Typora 中按 Ctrl+1~6 可快速切换标题层级。

2. 段落与换行

功能 源码写法 渲染效果
普通段落 直接写文本，段落间空一行 段落 1
段落2（空行分隔）
强制换行 行尾加 2 个空格 + 回车 或 <br> 第一行（2 个空格结尾
第二行（强制换行））

3. 文本强调（突出重点）

层级 源码写法 示例 渲染效果
斜体 *文本* 或 _文本_ *斜体内容* 斜体内容
粗体 **文本** 或 __文本__ **粗体内容** 粗体内容
粗斜体 ***文本*** 或 ___文本___ ***粗斜体体内容*** 粗斜体体内容
删除线 ~~文本~~ ~~删除内容~~ 删除内容
下划线 <u>文本</u>（HTML 标签） <u>下划线内容</u> 下划线内容

4. 列表（有序 / 无序 / 任务）

(1) 无序列表（无需排序）
源码写法： -/*/+ + 空格 + 文本（可嵌套，缩进 2 个空格）

示例

渲染效果

  • 前端技术栈
    • Vue 3.x
    • React（可选）
      • Next.js 框架
  • 后端技术栈
    • Java + Spring Boot

(2) 有序列表（需要排序）
源码写法： 数字.+ 空格 + 文本（序号自动递增，嵌套缩进）

示例

渲染效果

  1. 环境准备
    1. 安装 JDK 1.8+
    2. 安装 MySQL 8.0
  2. 项目启动
    1. 执行 SQL 脚本
    2. 启动应用

（3）任务列表（待办 / 已完成）
源码写法： - [ ] 未完成 / - [x] 已完成（中括号内空格 /x，符号后空格）

示例

渲染效果

  • 完成 Markdown 语法整理
  • 编写示例文档
  • 测试兼容性

5. 引用（标注说明 / 引用内容）

源码写法： > + 空格 + 文本（可嵌套，>> 二级引用）

示例

渲染效果

核心说明：本接口仅支持 POST 请求。

补充备注：请求头需携带 Token（有效期 2 小时）。

小贴士： Ctrl+Shift+Q 可快速插入引用。

6. 水平分隔线（分割章节）

源码写法： 单独一行输入 ---/***/___（前后空行避免冲突）

示例

渲染效果

第一章节

内容…

第二章节

内容…

二、链接与图片（文档跳转 / 配图）

1. 外部链接（跳转网页）

源码写法： [链接文本](链接地址 "可选 hover 标题")

示例

渲染效果

  • 官方文档：Vue.js
  • GitHub 仓库：Markdown 语法

2. 内部链接（文档内跳转）

源码写法： [链接文本](#目标标题)（目标标题需去掉特殊字符，空格替换为-）

示例

渲染效果： 点击目录链接可直接跳转到对应章节（长文档必备）

3. 图片（插入本地 / 网络图片））

源码写法： ![alt 文本](图片地址 "可选标题")（alt 文本为图片加载失败时显示，标题为鼠标悬浮显示）

示例

1. 行内代码（短代码片段）

源码写法： 代码内容（反引号包裹，避免与代码内字符冲突）

示例

渲染效果：
核心依赖：spring-boot-starter-web，执行命令：mvn clean package -DskipTests

多行代码块（长代码 / 配置）
