NVM的安装与配置

发表于2025-11-26|更新于2025-11-26|技术分享
|浏览量:

nvm（Node Version Manager）是管理 Node.js 版本的工具，支持在同一台设备上安装多个 Node.js 版本，并随时切换。

一、系统安装 nvm

步骤：

  1. 下载安装包
    进入 GitHub Releases，下载最新版的 nvm-setup.exe（或 nvm-setup.zip）。
  2. 运行安装程序
  • 选择 nvm 安装路径（建议默认 C:\Users\你的用户名\AppData\Roaming\nvm，避免中文 / 空格路径）。
  • 选择 Node.js symlink 路径（默认 C:\Program Files\nodejs，无需修改）。
  • 安装完成后，关闭现有命令行，重新打开。
  1. 验证安装
    执行命令 nvm version，显示版本号即安装成功

三、常用 nvm 命令（速查）

命令 功能说明 适用系统
nvm --version 查看 nvm 工具版本 所有
nvm version 查看 nvm 版本（部分系统别名，同 --version） 所有
nvm install <版本号> 安装指定 Node.js 版本（如 nvm install 18.17.1） 所有
nvm install lts 安装最新版 LTS 稳定版（长期支持版本） 所有
nvm install lts/<代号> 安装指定代号的 LTS 版本（如 nvm install lts/hydrogen 对应 Node.js 18.x） 所有
nvm use <版本号> 切换到指定已安装版本（Windows 下切换后自动设为默认） 所有
nvm alias default <版本> 设置默认 Node.js 版本（切换终端后仍生效） macOS/Linux
nvm use default 切换到默认版本 macOS/Linux
nvm ls 列出所有已安装版本（带 * 标记当前使用版本） 所有
nvm list nvm ls 的别名，功能一致 所有
nvm ls-remote 列出所有可安装的 Node.js 版本 所有
nvm ls-remote --lts 仅列出可安装的 LTS 稳定版 所有
nvm uninstall <版本号> 卸载指定已安装版本 所有
nvm current 查看当前正在使用的 Node.js 版本 所有
nvm node_mirror <url> 设置 Node.js 下载镜像（国内推荐 https://npmmirror.com/mirrors/node/） 所有
nvm npm_mirror <url> 设置 npm 下载镜像（国内推荐 https://npmmirror.com/mirrors/npm/） 所有
nvm proxy <url> 设置下载代理（空白查看当前代理，none 移除代理） 所有
nvm root <路径> 查看/设置 nvm 存储 Node.js 版本的根目录 所有
nvm arch 查看当前 Node.js 运行的架构（32 位/64 位） Windows
nvm on 启用 nvm 版本管理功能 Windows
nvm off 禁用 nvm 版本管理功能 Windows
